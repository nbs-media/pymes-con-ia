# PYMES con IA site

Static marketing site for PYMES con IA.

- Live domain: https://pymesconia.com
- Hosting: Vercel project `pgformula/pymes-con-ia`
- Routes:
  - `/`
  - `/privacidad`
  - `/terminos`

## Deployment

Best practice path is GitHub as source of truth, with Vercel linked to this repository for production deployments.

The local `.vercel/` directory is intentionally ignored. Use Vercel project settings/Git integration for deployment, not committed local project metadata.
